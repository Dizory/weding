# Дизайн приглашения

Поместите сюда JSON-экспорт из Figma.

## Варианты

### 1. В папку проекта (рекомендуется)

Положите файл сюда: **`design/figma-invitation.json`**

Затем напишите мне «готово» или «файл на месте» — я прочитаю его и реализую дизайн.

### 2. Figma Figment (плагин)

Если используете плагин Figment в Figma:
- Экспортируйте через «Real-time Export»
- Или сохраните в `~/.figma-exports/latest-figment.json`
- Затем скажите мне — я импортирую через `import_figma_figment`
